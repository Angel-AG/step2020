TwitterViz
A Twitter App to visualize how active a user is by plotting the frequency of Tweets, Replies and Retweets through time. Built with Python.
  • Used the Twython library to access and interact with Twitter APIs.
  • Built personalized data tables using Pandas by adding new columns and removing unnecessary data from Tweet objects.
  • Examined the resulting data using Seaborn by drawing count plots (bar plots) and line plots, with results showing the peaks and gaps of use in different periods of time.
